The app that i created intended use is to create a character by selecting the different
options from the drop down menus and then once all the fields are picked you press 
the create Character button
your stats will be generated below
you can then save the character or continue to customize
if you wish to look at the saved characters you can do that with the view roster button to see
the previously created characters
the view roster background is an image the in grabbed from the web

in order to build a working project you will need to install the following libraries as well as
replace the assets  folder that was provided

npm install @react-navigation/native @react-navigation/native-stack
expo install react-native-screens react-native-safe-area-context

expo install expo-sqlite

expo install @react-native-picker/picker

expo install expo-av
